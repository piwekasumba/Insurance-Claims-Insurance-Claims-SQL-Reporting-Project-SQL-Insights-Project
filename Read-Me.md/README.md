# Insurance Claims Risk Analytics SQL Insights Project

## Executive Summary

This project demonstrates how PostgreSQL can be used to analyse insurance claims data and produce reporting-ready information that supports business decision-making.

Using a simulated relational dataset, I applied SQL to organise data, answer business questions, calculate key reporting metrics, and identify trends that could assist operational reporting and claims monitoring.

The project reflects the practical SQL and analytical skills I am developing while preparing for an entry-level Reporting Analyst opportunity.

---

# Business Scenario

Insurance companies process large volumes of claims data every day.

To support effective decision-making, claims data must be organised into accurate, reliable reports that help monitor business performance, customer activity, and potential areas of risk.

This project demonstrates how SQL can transform raw business data into meaningful reporting outputs.
